# Mon site Personnel - Emmanuel Assi

Site vitrine dévéloppé avec Django, présentant mes projets et compétences.


## Déploiement
- **Hébergeur** : A déterminer
- **Domaine** : A achater
- **Statut** : En dévéloppemnt

## Fonctionnalités

- Potfolio de projets
- Blog personnel
- Formulaire de contact
- Design responsive moderne

## Technologies

- Django 4.2
- Python 3.8+
- Tailwind CSS
- HTMX

## Installation

1. Cloner le projet
2. 'pip install -r requirements.txt'
3. 'python manage.py migrate'
4. 'python manage.py runserver'
